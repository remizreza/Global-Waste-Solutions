## Vercel Auto Deploy (GitHub Actions)
Set these repository secrets to enable automatic production deploys on every push to `main`:
- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`
- `VERCEL_PROJECT_ID`

Workflow file: `.github/workflows/main.yml`.
